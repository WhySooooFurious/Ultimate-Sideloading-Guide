## [Post Install Notes](accent://)

Qualche linea guida da seguire per assicurarsi che le tue app non vengano revoked.

- Non disabilitare mail il Config Profile!

- Non abilitare 2 Config Profile allo stesso tempo

- Vai in Modalità Aereo se stai cambiando Config Profile!

- Disattiva WiFi e abilita la Modalità Aereo quando vuoi Spegnere, Riavviare e Aggiornare il dispositivo!

- Non puoi usare una VPN a meno che tu abbia seguito la VPN Setup guide!

- Controlla regolarmente Aggiornamenti, Annunci e Info sui Certificate su Portal!
